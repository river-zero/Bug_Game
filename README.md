## 벌레 게임
게임은 총 두 가지 단계로 구성되어 있습니다. 1단계는 사방으로 돌아다니는 벌레들을 마우스로 클릭해 모두 제거하면 클리어됩니다. 2단계는 마우스에 부착된 사탕을 다가오는 벌레들로부터 1분 동안 지켜내면 클리어됩니다. 해당 프로젝트는 Direct2D를 이용해 구현하였습니다.
